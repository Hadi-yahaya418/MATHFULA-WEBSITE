Algebra Formulas

1. Basic Algebraic Operations

  • Distributive Property:
    \[a(b + c) = ab + ac\]
  • Combine Like Terms:
    \[ax + bx = (a + b)x\]
  • Factoring:
    • Common Factor:
      \[ax + ay = a(x + y)\]
    • Difference of Squares:
      \[a^2 - b^2 = (a - b)(a + b)\]
    • Perfect Square Trinomial:
      • \(a^2 + 2ab + b^2 = (a + b)^2\)
      • \(a^2 - 2ab + b^2 = (a - b)^2\)
    • Quadratic Trinomial:
      \[ax^2 + bx + c = (mx + n)(px + q)\]

2. Exponents and Radicals

  • Laws of Exponents:
    • Product Rule:
      \[a^m × a^n = a^{m+n}\]
    • Quotient Rule:
      \[\frac{a^m}{a^n} = a^{m-n}\]
    • Power Rule:
      \[(a^m)^n = a^{m×n}\]
    • Zero Exponent Rule:
      \(a^0 = 1\) for \(a ≠ 0\)
    • Negative Exponent Rule:
      \[a^{-n} = \frac{1}{a^n}\]
  • Radicals:
    • Square Root:
      \[\sqrt{a} × \sqrt{b} = \sqrt{ab}\]
    • Product and Quotient Rules for Radicals:
      \[\sqrt{\frac{a}{b}} = \frac{\sqrt{a}}{\sqrt{b}}\]
    • Rationalizing the Denominator:
      \[\frac{a}{\sqrt{b}} = \frac{a \sqrt{b}}{b}\]

3. Linear Equations

  • Slope-Intercept Form:
    \[y = mx + b\]
  • Point-Slope Form:
    \[y - y_1 = m(x - x_1)\]
  • Standard Form:
    \[Ax + By = C\]
  • Slope Formula:
    \[m = \frac{y_2 - y_1}{x_2 - x_1}\]

4. Systems of Equations

  • Substitution Method: Solve one equation for one variable, then substitute into the other equation.
  • Elimination Method: Add or subtract equations to eliminate one variable.

5. Quadratic Equations

  • Standard Form:
    \[ax^2 + bx + c = 0\]
  • Quadratic Formula:
    \[x = \frac{-b ± \sqrt{b^2 - 4ac}}{2a}\]
  • Completing the Square:
    \[ax^2 + bx + c = 0\] can be written as \[a(x + \frac{b}{2a})^2 - \frac{b^2}{4a^2} + c = 0\]

6. Inequalities

  • Solving Linear Inequalities: Same steps as solving equations, but if you multiply or divide by a negative number, reverse the inequality sign.
    Example: \[-2x > 4 ⇒ x < -2\]
  • Compound Inequalities: Solve inequalities involving "and" & "or".
    \(a < x < b\) or \(x < a\) or \(x > b\)

7. Absolute Value

  • Absolute Value Definition:
    \(|x| = {cases} x\) & if \(x ≥ 0 -x\) & if \(x < 0 {endcases}\)
  • Solving Absolute Value Equations:
    \(|x| = a ⇒ x = a\) or \(x = -a\)
  • Solving Absolute Value Inequalities:
    • \(|x| < a ⇒ -a < x < a\)
    • \(|x| > a ⇒ x < -a\) or \(x > a\)

    • 8. Functions

    • Function Notation:
      \(f(x) = mx + b\)
    • Domain and Range:
      The domain is the set of all possible inputs (x-values), and the range is the set of all possible outputs (y-values).

    9. Sequences and Series

    • Arithmetic Sequence:
      \(a_n = a_1 + (n-1)d\)
    • Geometric Sequence:
      \(a_n = a_1 × r^{(n-1)}\)
    • Sum of an Arithmetic Series:
      \(S_n = \frac{n}{2} × (a_1 + a_n)\)
    • Sum of a Geometric Series:
      \(S_n = a_1 × \frac{1 - r^n}{1 - r}\) (if \(r ≠ 1\))
